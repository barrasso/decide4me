CCTransition Class Reference

Inherits from CCScene : CCNode : CCResponder : NSObject
Declared in CCTransition.h

Overview

Creates a transition. The transition will replace the outgoing scene, with the incoming scene. Both scenes can be animated during the transition ( default disabled ).

Complex animated scenes on retina devices might run slow during transitions. In that case, try disabling retina transitions, and check if the render quality is adequate.
